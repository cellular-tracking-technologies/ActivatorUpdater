1. Download both the ActivatorUpdater.zip and ActivatorFirmware.hex files.
2. Unzip the ActivatorUpdater folder and run ActivatorProgrammer.exe
3. Plug the Activator in to the computer using the USB cable.
4. Click on the […] button next to the HEX File text box and select the ActivatorFirmware.hex file.
5. Click on the dropdown box next to COM Port and select the proper COM port.
6. Click Program.
7. A screen should pop-up with information about the update.
8. When it’s completed there will be a pop-up dialog stating that the update is complete.
